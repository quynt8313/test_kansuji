# Homework

Create library to convert kanji and number  

## Example 
- 1.to_kansuji #=> "一"
- "一".to_number #=> 1

# Enviroment 

Install ruby2.3 or later (recommend rbenv)

$ bundle install

# Evaluate 

pass ↓
```sh
$ bundle exec rspec
```
